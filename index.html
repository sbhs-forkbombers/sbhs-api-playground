The code may be found here.

Timetable APIs

  • /api/timetable/daytimetable.json
    - today's timetable including changed rooms/teachers (raw SBHS data). Docs
  • /api/timetable/timetable.json
    - full timetable, not real-time. Docs
  • /api/timetable/bells.json
    - today's schedule. Used by sbhsbelltimes.tk and sbhstimetable.tk

Daily news API

  • /api/dailynews/list.json
    - daily notices (mango sheet). Docs

User Details

  • /api/details/participation.json
    - a user's award scheme points information. Docs
  • /api/details/userinfo.json
    - a user's information. Docs

Custom APIs

These are wrappers around the existing APIs. They provide data in a more convenient form for the client. 
  • /api/today.json
    - a wrapper around 
    /api/daytimetable.json
  • /api/notices.json
    - a wrapper around 
    /api/notices.json

Session Manipulation/Display

  • /session_info
  • /logout
  • /reset_access_token
  • /refresh_token
    - use the refreshToken to acquire a new access code.
